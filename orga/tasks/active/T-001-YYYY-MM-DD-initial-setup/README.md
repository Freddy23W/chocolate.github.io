# T-001 Queer Art for Care Festival Website Setup

Status: active
Created: 2025-03-05

## Goal

- Create a landing page for the Queer Art for Care festival that automatically plays audio content when accessed via QR codes on chocolate bars.

## Scope

In:
- MP3 audio file ("Chocolate - The Podast Ep.1 (Unsweetend Chocolate) .mp3")
- Windsurf workflow template structure
- Basic web technologies (HTML/CSS/JavaScript)

Out:
- Responsive landing page with automatic audio playback
- Project documentation and structure
- Deployment-ready static website

## Acceptance criteria

- Landing page loads successfully when QR code is scanned
- Audio plays automatically (or with single tap on mobile)
- Design is mobile-responsive and reflects queer festival aesthetics
- Page works on major browsers (Chrome, Safari, Firefox)
- Audio controls are visible and functional
- Fast loading (<3 seconds)
